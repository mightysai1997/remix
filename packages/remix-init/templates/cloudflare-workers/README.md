## Development

From your terminal:

```sh
npm run dev
```

This will build your app in development mode and start a remotesession with cloudflare,
rebuilding assets on file changes.

## Deployment

Use wrangler to build and deploy your application to Cloudflare Workers:

```sh
wrangler publish
```
