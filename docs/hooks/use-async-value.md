# `useAsyncValue`

Returns the resolved data from the closest `<Await>` ancestor component.

```tsx
function SomeDescendant() {
  const value = useAsyncValue();
  // ...
}
```

```tsx
<Await resolve={somePromise}>
  <SomeDescendant />
</Await>
```
